# project_explorer

## Keyboard Shortcuts
**Important Note:** The Delete action deletes the selected files/folders immediately without confirmation or sending to trash!

| Action | Keys                                              |
| ------ | ------------------------------------------------- |
| Trash  | (<kbd>Del</kbd>)                                  |
| Delete | (<kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>Del</kbd>) |
